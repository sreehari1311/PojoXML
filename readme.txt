========
PojoXML
========

This project is migrated from code.google.com , its an old project which which has been tested on java 1.5 . 

**********New Version is under developemnt *****************


version 1.0 

 	PojoXML library provides flexible way to convert  POJO to xml file and convert xml file back to POJO object.
Xml generation is based on the private fields declared in the java bean object. PojoXML invokes its getter method and generates 
xml tags and data based on the instance variable name and its value.
